# 实验结论

## 结论摘要
仅将批处理聚合窗口从 8 增大到 32,p99 延迟从 118.4ms 降至 96.2ms(−18.8%),
吞吐从 2140 升至 2612 req/s(+22.1%),首次进入 SLA 达标线(p99 ≤ 100ms)。
代价是 p50 从 42.1ms 升至 47.3ms(+12.4%)。

- 完成日期:2025-11-06
- 执行时长:约 2.5 小时(5 轮压测 55 分钟 + 分析与文档)

## 结论适用范围
本结论仅在以下条件下成立,禁止无限定泛化:
- 环境:Ubuntu 22.04 / Python 3.11.8 / PostgreSQL 14.11 本机部署 / 连接池上限 20 / 服务 commit 9f31c2e
- 负载:orders-replay-50k.bin 录制回放,200 并发,10 分钟,无低谷段(低谷时段的窗口退化未验证)
