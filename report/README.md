# report/

本目录用于沉淀最终方案与最终报告,是整个工作流的终点落点。

---

## 何时进入最终报告(终止条件白名单)

仅当命中以下条件之一,且能引用证据时,才进入最终整理阶段:

1. **已达标**:核心指标满足 `problem/scoring-and-sla.md` 的达标条件(引用 EVDxxx)。
2. **稳定持平**:连续多个实验的改进低于噪声阈值,且已回到调研阶段补充过假设仍无进展(引用相关 Exxx)。
3. **外部原因**:外部规则或环境变化使继续优化无意义(记 Dxxx)。
4. **理论极限**:有证据表明已接近该环境下的理论上限(引用分析与 EVDxxx,以及相关实验 plan.md 的"理论上界估算"节——无上界估算不得以此为由终止)。

未命中任何条件时,应回到工作流继续迭代(修订假设或补充调研),而不是提前收敛。
